Gagal Memuat Data

Tidak dapat mengambil data dari Google Sheets. Pastikan URL CSV sudah benar dan spreadsheet sudah dipublikasikan ke web.

Cara fix: Google Sheets → File → Share → Publish to web → pilih CSV
